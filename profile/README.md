# Plan4Land 소개

<h3 style="border-bottom: none;">국내 관광지 검색 및 여행 플래닝 작성 플랫폼</h3>

- 관광지, 맛집, 숙소 정보를 한 곳에서 검색 가능하도록 지원

- 사용자들이 함께 여행 계획을 세울 수 있도록 지원  

- 다른 사람들의 여행 계획을 참고하여 나만의 여행을 쉽게 구상  

- 기차 및 고속버스 운행 정보를 실시간으로 확인 가능
